# ✍️ Coaching AI Tools for Coding and Data Tasks

**NICAR 2026 · Thursday, March 5 · 11:30 a.m.**

Charles Minshew, The Atlanta Journal-Constitution

## What we'll cover

- Drafting public records requests with AI (using Indiana's APRA as our example)
- Understanding sample data: data dictionaries, scope, story questions
- Getting advice for Excel tasks — then translating to R or Python
- Debugging scripts with AI tools
- Prompt patterns you can reuse back at your newsroom

## Tipsheet

**[View the online tipsheet →](https://charlesminshew.github.io/nicar26/coaching-ai-tools/)**

The tipsheet includes:
- Ground rules: what to use AI for (and what not to)
- Three things to try your first day back in the newsroom
- Six reusable prompt patterns with example prompts
- The full records request workflow (specific → iterate → structured data → verify)
- Tool comparison notes (ChatGPT, Claude, Gemini, Copilot)
- Resource links

## What you'll need

- A ChatGPT account (free or paid) — we'll use this for hands-on exercises
- Other AI tools welcome: Claude, Gemini, Copilot, etc.
- An interest in data and open records
- Ability to argue with a generative AI service
