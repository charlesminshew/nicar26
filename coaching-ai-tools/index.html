Coaching AI Tools for Coding, Data & Open Records

The ground rules

✅ Do use ChatGPT for…

  • Writing & debugging code
  • Understanding unfamiliar data
  • Drafting public records requests
  • Internal documentation
  • Taking better notes for yourself
  • As a super-powered search engine

🚫 Don't use ChatGPT for…

  • Final data analysis work
  • Handling sensitive personal data
  • Processing entire datasets
  • Writing stories or content for the public
  • Replacing the work of other team members
  • Making editorial decisions

Important: These guidelines are just suggestions. Consult with your bosses or attorney for more specific recommendations.

After NICAR: What to try first

1. Draft a records request

Pick an agency you cover. Start with a specific address or entity. Ask AI to draft the request under your state's public records law. Then iterate: expand the scope, ask for structured data, and request specific field names.

2. Understand a new dataset

Upload a sample (not the full file) to ChatGPT or Claude. Ask for a data dictionary. Ask "What questions could a reporter ask with this data?" Then ask for code to explore it.

3. Debug or write code

Paste your error message along with your code. Describe what you expected vs. what happened. Ask it to explain the fix — not just give you the fix.

Crafting effective prompts

🔹 Clear & Direct

These show the range of what you can ask. Pick one and try it right now.

Creative

Imagine that Taylor Swift wrote a song about data journalism (set it to the tune of "Enchanted").

Format-shifting

Write three essential tips for data journalists in the style of a simple, witty and memorable highway sign.

Constrained

Write a haiku about filing an open records request using Indiana's APRA.

Practical

Show me how to write an R script that filters my dataset of crashes in the Indianapolis metro area last year.

If you're comfortable with coding.

🔹 Be Specific → Be More Specific

The more context you give, the better the output. Compare these two prompts:

Be Specific

Write a records request for elevator inspection reports at 10 S West St, Indianapolis, IN.

Be More Specific

You're a reporter and you need information about elevator inspections at the JW Marriott at 10 S West Street, Indianapolis, IN. Write a records request seeking reports from the Indiana Department of Homeland Security.

🔹 Iterate

Don't stop at the first response. Push it further with follow-up prompts.

Iterate

You're a reporter and you need to put your elevator reports from the JW Marriott into context. Write a records request to the Indiana Department of Homeland Security for inspection data.

Now we're going beyond one building to get the bigger picture.

Iterate some more

Write an open records request to the Indiana Department of Homeland Security for a state database of elevator inspection reports since January 1, 2020.

Now we're asking for structured data with specific parameters. This is the power move.

More prompt patterns to bookmark

These are reusable structures you can adapt for almost any task.

Role + Task

You are a data reporter covering public safety in Indiana. Help me write an APRA request for elevator inspection records statewide.

Assigning a role helps the AI calibrate tone and domain knowledge.

Show me, then explain

Write an R script that reads this CSV, calculates a rolling 7-day average of daily counts, and plots it. Then explain each step so I can modify it.

Gets you working code AND understanding.

Ask for structured output

Give me this data as a CSV with columns for: date, location, inspection_type, result, inspector_name, deficiencies_noted.

Specifying the output format prevents vague, narrative-style answers.

Sanity check

What assumptions are you making in this analysis? What could be wrong? What would a skeptical editor ask about these numbers?

AI won't challenge itself unless you ask. Build this into your workflow.

Data dictionary

Here's a sample of 20 rows from this dataset. Write a data dictionary explaining each column: what it contains, the data type, and any patterns or issues you notice.

Upload a sample — never the whole dataset.

Verify the AI's work

Check the statute citation you just used. Is Indiana Code § 5-14-3 the correct reference for APRA? What's the full citation?

AI will hallucinate legal citations. Always double-check.
