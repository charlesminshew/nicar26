# 🕷️ Building Web Scrapers on the Fly with ChatGPT and Claude

**NICAR 2026 · Friday, March 6 · 3:45 p.m.**

Charles Minshew, The Atlanta Journal-Constitution

## What we'll cover

- Identifying scrapable data on a web page
- Prompting AI to write scraping code (Python and JavaScript)
- Handling common challenges: pagination, dynamic content, authentication walls
- Cleaning and structuring scraped output
- When to scrape vs. when to look for an API or bulk download
- Ethics and legal considerations
